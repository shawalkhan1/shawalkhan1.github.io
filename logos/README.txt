REAL LOGOS — DROP THEM IN HERE
==============================

The site shows clean text monograms by default. To replace a monogram with the
real logo, add an image file to THIS folder with the EXACT name below. The page
detects the file automatically and swaps it in — you do NOT edit any code.

Filenames the site looks for:

  lums.png      ->  LUMS (education + experience host institution)
  comsats.png   ->  COMSATS University
  awaaz.png     ->  Awaaz-e-Sehat
  gates.png     ->  Bill & Melinda Gates Foundation
  aihub.png     ->  National AI Hub (Pakistan's first AI platform for MNCH)

Tips:
- PNG with a transparent background looks best (SVG also works — if you use SVG,
  keep the same name but .svg, e.g. lums.svg, and update the src in index.html).
- Roughly 120x120 px (or any square-ish size) is plenty; the site scales them.
- Use official logos only where you're permitted to (these mark genuine
  affiliations, which is standard on a personal site/CV).
- If you don't add a file, the tasteful monogram simply stays — nothing breaks.
